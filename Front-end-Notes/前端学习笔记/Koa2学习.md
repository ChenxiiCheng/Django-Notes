# Koa2学习

1. **安装koa2**

   ```
   sudo npm install -g koa-generator
   ```

   

2. **创建项目**

   ```
   koa2 -e koa2-learn
   ```



3. 进入koa2目录

   ```
   cd koa2-learn
   
   #install dependencies:
   npm install
   
   #run the app: 一般用第二种启动：
   DEBUG=koa2-learn:* npm start
   
   #第二种启动方式：
   npm run dev
   
   #页面地址： localhost:3000
   ```
